Tiêu đề trang web

CSS viết tắt của Cascading Style Sheets là một ngôn ngữ định dạng trang web dùng để mô tả cách hiển thị các phần tử HTML trên màn hình, trên giấy hoặc trên các thiết bị khác. CSS giúp bạn kiểm soát được vị trí, cỡ chữ, màu sắc và các thuộc tính khác của các phần tử HTML trên trang web của mình.
